Плагин агрегации комментариев

== Description ==

Плагин служит связующим звеном агрегации комментариев в одном рабочем пространстве для последующиего их модерирования. Рабочее пространство предоставляется отдельно и устанавливается на собственный сервер.

== Installation ==

1. Загрузите "complag" в "/wp-content/plugins/".
2. Активируйте плагин на странице "Плагины" в меню WordPress.
3. Вставьте сгенерированный ключ (генерируется на удаленном сервере) в поле настройки плагина.
